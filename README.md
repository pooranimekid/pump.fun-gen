## Prerequisites

- Node.js
- NPM
- Basic terminal knowledge

## Setup

1. Clone repo
2. Install deps with `npm install` 
3. Make auth.yaml file
4. Start with `node index.js`

## Features

- Auto email gen
- Auto verify
- Token storage
- Non-stop account creation
- Error handling

## Warning

## How it works

This tool:
- Makes temp emails
- Creates accounts
- Gets verify codes
- Saves auth tokens
